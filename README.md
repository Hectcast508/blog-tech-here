# blog-tech-here

## Description

Using this application will allow everyone to signup and start posting blogs about tech. You are able to have descussions by commenting on a blog that interest you.

## Installation

```
npm install
```

MySQL

```
source db/schema.sql
```

Command line

```
npm run seeds
```

```
npm start
```
